# 7 Colors
<p align="center"><img src="https://raw.githubusercontent.com/WerWolv/EdiZon/master/icon.jpg"](https://raw.githubusercontent.com/AmauryZ/7colors_switch/refs/heads/main/Filler-1990.jpg"></p>

## Controls

| Letter | Key |
|--------|-----|
| A | A |
| B | B |
| C | Y |
| D | X |
| E | DPad Left |
| F | DPad Up |
| G | DPad Right |
